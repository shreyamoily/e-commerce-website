# restaurant-website

This is a basic Restaurant website implemented with HTML, CSS and Javascript. A user can access wide variety of 
mouthwatering dishes to look at and order the one he finds delicious and interesting. A user can also have a look at 
the restaurant's interior and reserve a table to have a wonderful experience.

## Features
- Menu
- Reservation of table
- Gallery
- Contact
